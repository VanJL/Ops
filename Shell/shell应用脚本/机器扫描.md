# 内网IP地址扫描

> 本章主要介绍如何对局域网中所有的机器IP地址进行扫描，快速的查找出局域网内存在哪些机器（哪些能通的机器）

## 扫描思路

1. 指定网段，多个网段
2. 计算并罗列出指定网段内所有IP地址
3. ping所有ip地址并对结果进行分析：如果ping不通的ip丢弃，能ping通的机器判断ttl是否小于等于64，小于等于64则留下，大于则丢弃或者记录到其他系统。
4. RHEL系统，也要区分5/6/7。
